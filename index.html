Traveling Salesman Heuristic Comparison

CSC 512 Project

The Traveling Salesman Problem is a well-known problem in combinatorial optimization. Given a complete graph, the solution takes the form of the shortest path that will visit each node exactly one time, returning to the first node at the final step. /br /br This project's goal was initially to compare several leading algorithms that approximate the optimal solution, including one modeling the tour as a quantum system. This project is still in progress, but several heuristics have been completed (if not completely optimized), including several construction heuristics, a simulated annealing heuristic, and an adaptation of the Lin-Kernighan algorithm.
